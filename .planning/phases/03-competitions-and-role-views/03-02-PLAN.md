   - `getTeamHeatmap()`: SELECT u.id AS "userId", u.name, u.team_id AS "teamId", MAX(a.created_at) AS "lastActivityAt" FROM users u LEFT JOIN activities a ON a.user_id = u.id AND a.created_at >= date_trunc('day', NOW() AT TIME ZONE 'UTC') WHERE u.role = 'rep' GROUP BY u.id, u.name, u.team_id ORDER BY u.name
   - Return type: `{ userId: number; name: string; teamId: number; lastActivityAt: string | null }[]`
   - Status derivation happens client-side (NOT in query): lastActivityAt within 30 min of Date.now() = green, same day but older = amber, null = red
