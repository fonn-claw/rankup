1. **Create src/lib/queries/admin.ts** -- Follow raw SQL pattern:
   - `getTeamStats()`: SELECT t.id, t.name, COUNT(u.id) as "repCount", COALESCE(SUM(a.xp_earned), 0)::int as "totalXp", COALESCE(SUM(a.xp_earned)/NULLIF(COUNT(DISTINCT u.id), 0), 0)::int as "avgXp" FROM teams t LEFT JOIN users u ON u.team_id = t.id AND u.role = 'rep' LEFT JOIN activities a ON a.user_id = u.id GROUP BY t.id, t.name ORDER BY "totalXp" DESC. Also LEFT JOIN battles to get active battle status for each team: check if team is in any battle WHERE status = 'active' and dates are current. Return: `{ id: number; name: string; repCount: number; totalXp: number; avgXp: number; inActiveBattle: boolean }[]`
   - `getEngagementData(days: number = 30)`: SELECT date_trunc('day', a.created_at)::date as "day", COUNT(DISTINCT a.user_id)::int as "activeReps", COUNT(*)::int as "totalActivities" FROM activities a WHERE a.created_at >= NOW() - make_interval(days => ${days}) GROUP BY "day" ORDER BY "day" ASC. Return: `{ day: string; activeReps: number; totalActivities: number }[]`
   - `getSettings()`: SELECT key, value FROM settings. Return as Map<string, string> for easy lookup.
