   - `getActiveBattle()`: SELECT battle with joined team names WHERE start_date <= NOW() AND end_date >= NOW() AND status = 'active', ORDER BY start_date DESC LIMIT 1. Return null if no rows. Return type: `{ id: number; team1Id: number; team2Id: number; team1Name: string; team2Name: string; startDate: string; endDate: string; status: string }`
   - `getBattleXp(team1Id, team2Id, startDate, endDate)`: CTE aggregating SUM(a.xp_earned) per user within date range for both teams, then GROUP BY team_id returning teamId, teamTotal (int), and members JSON array (userId, name, xp) ordered by xp DESC. Use `>=` for startDate and `<=` for endDate to avoid off-by-one. Return type: array of `{ teamId: number; teamTotal: number; members: { userId: number; name: string; xp: number }[] }`

4. **Create src/lib/queries/challenges.ts**:
   - `getActiveChallenges(userId: number)`: SELECT active challenges (start_date <= NOW() AND end_date >= NOW()) LEFT JOIN challenge_participants on challenge_id and user_id. Return: `{ id: number; title: string; description: string | null; goalType: string; goalValue: number; endDate: string; progress: number }[]`. COALESCE progress to 0.

5. **Create src/lib/hooks/useCountdown.ts** -- 'use client' hook:
   - `useCountdown(endDate: string)` returns `{ days: number; hours: number; minutes: number; expired: boolean }`
   - Uses useState + useEffect with setInterval at 60_000ms (60 seconds)
   - When remaining < 1 hour, switch interval to 1_000ms (1 second) for per-second precision
   - `computeRemaining(endDate: string)` helper: diff = new Date(endDate).getTime() - Date.now(), derive days/hours/minutes, if diff <= 0 return expired: true
